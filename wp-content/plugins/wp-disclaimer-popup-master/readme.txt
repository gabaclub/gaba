 == Description ==
Creates a disclaimer popup on the page using FancyBox where the short code is used. 

Each disclaimer is saved as a custom post type with options for changing the "Accept", "Decline", "Redirect" and "Time to hold". 

If a user accepts it will save that in a cookie for the amount of time that was mentioned.

This plugin is available on github also at https://github.com/artarmstrong/wp-disclaimer-popup

== Installation ==
1. Install the plugin
2. Activate the plugin
3. Create a new disclaimer from the "Disclaimers" menu item
4. Place the short code referencing the disclaimer you would like to use in a page.
5. Thats it!
